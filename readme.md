<h1> Frogger Game</h1>


<ul>

<h2> Instructions </h2>

<br>


<li style="color:red;">

Z or Up arrow to move forward 


</li>


<li style="color:green;">

S or Down arrow to move back


</li>


<li style="color:blue;">

Q or Left arrow to move left


</li>

<li style="color:yellow;">

D or Right arrow to move right


</li>

</ul>
